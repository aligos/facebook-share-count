Plugin that gives you shortcodes and PHP functions to count posts/pages shares on 6 social networks!

== Description ==
= About =
The plugin function is for get and show total of facebook shares for a given post by post/page ID.

== Screenshots ==
1. Shares count in admin edit post view
2. Shares count in admin edit post/pages/post types list view

== Installation ==
1. Go to your admin area and select Plugins -> Add new from the menu.
2. Search for \"Facebook Share Count\".
3. Click install.
4. Click activate.
